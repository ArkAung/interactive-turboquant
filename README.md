# TurboQuant — a first-principles walk-through

An interactive, single-file explainer for Google Research's [TurboQuant](https://arxiv.org/abs/2504.19874) algorithm — the rotation trick that compresses AI vectors to ~3 bits per number with near-Shannon distortion.

Open `index.html` in a browser. No server, no build step — just HTML, CSS, JS, a sprinkle of Canvas 2D, and MathJax from a CDN.
